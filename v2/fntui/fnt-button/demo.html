type

展示类型

circle 按钮呈现边缘圆形的样式

+ Type circle

text 按钮呈现文本样式

Type theme text

outlined 按钮呈现边框模式，默认情况下为白色，需要设定theme可改变颜色

Type outlined Type outlined2

Disabled button
