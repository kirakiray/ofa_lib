o-code

如何使用 o-code？

该组件依赖 ofajs ，请先将 ofajs 引用到项目内；

项目内添加 ofa.js 后，使用 load 方法加载 o-code组件；

o-code是干什么用的？

过去，想要在web上展示代码片段，使用的是 code 标签；

如下展示弹窗的html代码

使用代码
显示效果
<script>alert("弹窗")</script>

显示效果无高亮颜色提醒，无格式化，阅读性特别差；而且还要对标签代码进行转义；

使用 o-code 来展示代码

如果使用o-code，将会得到更易读的显示效果；

（代码片段必须使用 template标签包裹 ）

使用代码
显示效果

和原生code标签对比，o-code的优势如下：

  • 有高亮颜色效果，采用vscode的默认高亮配色；
  • 会自动格式化代码，提高阅读性；
  • 不需要转义html的代码；

o-code支持的属性

lang

目前支持

o-code主要使用 highlightjs添加样式，所以只要highlightjs支持的，o-code都支持；

  • html(默认)
  • javascript
  • css
  • python
  • bash
  • Markdown
  • ...
